# Radii

Шкала радиусов скруглений. Используется единая шкала на весь продукт.

## Tokens

| Token       | Value | Когда                                              |
| ----------- | ----- | -------------------------------------------------- |
| `--r-xs`    | 2px   | очень мелкие чипы (KIND_LABEL), state-pill, hairline-карточки |
| `--r-sm`    | 4px   | поля ввода, мелкие кнопки, popover-item, editable-title |
| `--r-md`    | 6px   | основные кнопки, icon-btn, sb-item, role-card, popover-icon |
| `--r-lg`    | 8px   | карточки (`.card`), модалки, попоперы, sd-domain   |
| `--r-xl`    | 10px  | приподнятые KPI-tile, contained-blocks внутри hero |
| `--r-pill`  | 999px | таблетки (avatars, dots, suggest-chip, create-chip, кнопки-таблетки) |

## Rules

- **Капсульная форма** (полностью круглые торцы) — `999px`, не `50%`. Использование
  `50%` ломает форму на не-квадратных элементах.
- **Иерархия**: если карточка `--r-lg` содержит inputs — у них `--r-sm`. Inner
  меньше outer.
- **Не миксуйте 4 значения на одном экране** — обычно достаточно `--r-md` для
  интерактивных элементов + `--r-lg` для контейнеров.
- **Bubble чата** — `12px` с одним прижатым углом (`border-bottom-right-radius: 4px`
  для user, `border-bottom-left-radius: 4px` для bot). Это особый паттерн,
  токенизировать не имеет смысла.
