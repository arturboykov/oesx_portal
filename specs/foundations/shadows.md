# Shadows

Тени используются скупо — основная иерархия плоскостей строится через
`--surface-*` подложки, а не через тени. Тени появляются только там, где
элемент **физически отрывается от плоскости**: модалки, попоперы, тосты,
drawer.

## Tokens

| Token                  | Value                              | Когда                          |
| ---------------------- | ---------------------------------- | ------------------------------ |
| `--shadow-card-light`  | `0 1px 4px rgba(0,0,0,0.06)`       | редко — light theme карточки   |
| `--shadow-glass`       | `0 1px 4px rgba(0,0,0,0.30)`       | элементы с backdrop-filter     |
| `--shadow-glow-teal`   | `0 0 24px rgba(29,184,154,0.15)`   | акцентный glow primary CTA     |

## Inline (не-токенизированные) тени для оверлеев

Эти тени применяются разово и не выносятся в токены — слишком специфичны.

| Element            | Shadow                                                                  |
| ------------------ | ----------------------------------------------------------------------- |
| `.modal`           | `0 24px 64px rgba(0,0,0,0.5)`                                          |
| `.popover` (dark)  | `0 12px 32px rgba(0,0,0,0.42), 0 2px 6px rgba(0,0,0,0.25)`             |
| `.popover` (light) | `0 8px 24px rgba(14,122,104,0.14), 0 2px 4px rgba(0,0,0,0.06)`         |
| `.toast` (dark)    | `0 12px 32px rgba(0,0,0,0.40), 0 2px 6px rgba(0,0,0,0.25)`             |
| `.toast` (light)   | `0 8px 24px rgba(14,122,104,0.18), 0 2px 4px rgba(0,0,0,0.08)`         |
| `.chat-history-drawer` (dark)  | `4px 0 24px rgba(0,0,0,0.30)`                              |
| `.chat-history-drawer` (light) | `4px 0 24px rgba(14,122,104,0.10)`                         |
| Sidebar overlay <1200px (dark) | `4px 0 24px rgba(0,0,0,0.30)`                              |
| Tooltip            | `0 6px 16px rgba(0,0,0,0.32)`                                          |

## Glow (focus/hover acent)

Точки состояния (`.dot-pos`, статус-пилюли) получают мягкий glow:

```css
.dot-pos { box-shadow: 0 0 6px rgba(29,184,154,0.55); }
```

Это сигнал «активно», не общий стиль теней.

## Rules

- **Не дублируйте тени на вложенных элементах** — если модалка уже отбрасывает
  тень, попоперы внутри неё могут обходиться без тени (или иметь minimal hint).
- **Цвет тени в светлой теме** — `rgba(14,122,104,…)` (тёмно-бирюзовый), не
  чисто-чёрный. Это сохраняет цветовое единство и не выглядит грязно.
- **На карточках по умолчанию НЕТ тени** — карточка определяется бордером
  `0.5px solid var(--border)`, не тенью. Тень добавляется только при `hover`
  у интерактивных карточек (`transform: translateY(-2px)`).
